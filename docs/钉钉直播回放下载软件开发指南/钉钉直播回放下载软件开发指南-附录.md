# 术语表

## A

**API (Application Programming Interface)**
应用程序编程接口,定义软件组件之间交互的规范和协议。

**Authentication**
身份认证,验证用户或系统身份的过程。

**Authorization**
授权,确定已认证用户或系统可以访问哪些资源的过程。

## C

**Cookie**
HTTP Cookie 是服务器发送到用户浏览器并保存在本地的一小块数据,用于在后续请求中识别用户。

**Cookie Jar**
Cookie 容器,用于存储和管理多个 Cookie 的数据结构。

## D

**Download Manager**
下载管理器,负责管理文件下载过程的组件,包括任务调度、进度跟踪、错误处理等。

**DingTalk**
钉钉,阿里巴巴集团开发的企业级智能移动办公平台。

## E

**Error Code**
错误码,用于标识特定错误类型的数字或字符串代码。

**Exponential Backoff**
指数退避,一种重试策略,每次重试的等待时间按指数增长。

## F

**FFmpeg**
一个开源的多媒体处理框架,可以记录、转换和流式传输音频和视频。

**Fixture**
测试夹具,测试框架中用于提供测试所需的数据或环境的机制。

## H

**HTTP (Hypertext Transfer Protocol)**
超文本传输协议,用于在 Web 浏览器和服务器之间传输数据的应用层协议。

**HLS (HTTP Live Streaming)**
苹果公司开发的基于 HTTP 的流媒体网络传输协议。

## I

**Integration Test**
集成测试,测试多个组件或模块之间交互的测试方法。

## J

**JSON (JavaScript Object Notation)**
一种轻量级的数据交换格式,易于人阅读和编写,同时也易于机器解析和生成。

## L

**Logging**
日志记录,将应用程序运行时的事件和状态信息记录到日志文件的过程。

**Log Level**
日志级别,用于标识日志消息重要程度的级别,如 DEBUG、INFO、WARNING、ERROR、CRITICAL。

## M

**m3u8**
HLS 协议使用的播放列表文件格式,包含媒体分片的 URL 信息。

**Mock**
模拟对象,在测试中用于替代真实对象的测试替身。

**Module**
模块,包含相关功能和代码的 Python 文件。

**Mypy**
Python 的静态类型检查工具。

## N

**N_m3u8DL-RE**
一个开源的 m3u8 下载工具,支持多种功能和特性。

## O

**OAuth**
开放授权,一种安全的授权框架,允许用户授权第三方应用访问其资源。

## P

**PEP 8**
Python Enhancement Proposal 8,Python 代码风格指南。

**Pytest**
Python 的测试框架,提供简单灵活的测试功能。

**Performance Test**
性能测试,评估系统性能指标的测试方法。

## R

**Retry Mechanism**
重试机制,在操作失败后自动重试的策略。

**Request**
请求,客户端向服务器发送的 HTTP 消息。

**Response**
响应,服务器向客户端返回的 HTTP 消息。

## S

**Segment**
分片,视频或音频文件被分割成的小块。

**Session**
会话,用户与系统交互的持续时间段。

**Static Type Checking**
静态类型检查,在代码运行前检查类型错误的工具或过程。

## T

**TS (Transport Stream)**
传输流,一种用于音视频传输的容器格式。

**Thread**
线程,进程中的执行单元。

**Timeout**
超时,操作等待的最长时间。

**Type Annotation**
类型注解,Python 3.5+引入的类型提示语法。

**Type Hint**
类型提示,用于指示变量、函数参数和返回值类型的注释。

## U

**Unit Test**
单元测试,对软件中最小可测试单元(通常是函数或方法)进行验证的测试方法。

**URL (Uniform Resource Locator)**
统一资源定位符,用于标识互联网上资源的地址。

**URL Parser**
URL 解析器,用于解析和分析 URL 的组件。

## V

**Video Merger**
视频合并器,将多个视频分片合并为完整视频的工具或组件。

## Y

**YAML**
一种人类可读的数据序列化格式,常用于配置文件。
